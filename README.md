# Sistema NAPI Rotas

### Divisão
/backend 
/frontend

### Desenvolvimento

### Back End
Requisitos
- Java 21
- MySQL
- Código do projeto

```bash
git clone <repositorio>
cd backend
./gradlew bootRun
```

### Front End
Requisitos
- Node.js 20+ (recomendado)
- npm ou yarn
- Código do projeto

```bash
git clone <repositorio>
cd frontend
npm install
npm install react-router-dom
npm install leaflet react-leaflet
npm run dev
```
